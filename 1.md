# 🚀 Hướng dẫn cài đặt & chạy

## Bước 1: Tạo môi trường ảo

```bash
cd voice-assistant
python -m venv venv
venv\Scripts\activate        # Windows
```

## Bước 2: Cài thư viện

```bash
pip install -r requirements.txt
```

> ⚠️ `sounddevice` trên Windows cần **PortAudio**.
> Nếu lỗi khi import, chạy: `pip install sounddevice --extra-index-url https://files.pythonhosted.org/packages`
> Hoặc cài thủ công: https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio

## Bước 4: Lấy Gmail App Password

1. Vào https://myaccount.google.com/security
2. Bật **2-Step Verification** (nếu chưa bật)
3. Tìm **App Passwords** → Tạo mới
4. Chọn "Mail" + "Windows Computer"
5. Copy 16 ký tự vào `.env`

## Bước 5: Chạy chương trình

```bash
python main.py
```

## Troubleshooting

| Lỗi | Nguyên nhân | Cách xử lý |
|-----|-------------|------------|
| `No module named sounddevice` | Chưa cài | `pip install sounddevice` |
| `PortAudioError` | Mic không nhận | Kiểm tra mic trong Device Manager |
| `SMTPAuthenticationError` | Sai App Password | Tạo lại App Password |
| `Model not found` | Chưa download | Lần đầu sẽ tự download (~250MB cho small) |
